MetVUW Plus — Privacy Policy

Last updated: March 2026

MetVUW Plus is a Chrome browser extension that adds usability enhancements to metvuw.co.nz. This policy explains what data the extension stores and how it is used.

Data collected

MetVUW Plus stores only the following user interface preferences:

  • Preferred forecast region (e.g. "nz", "nzsi")
  • Preferred forecast type (e.g. "rain", "wind850")
  • Preferred number of forecast days
  • Whether automatic redirection to the preferred view is enabled

These preferences are stored using the Chrome chrome.storage.sync API, which may synchronise them across devices signed in to the same Google account. No other data is collected, transmitted, or shared.

Data not collected

MetVUW Plus does not collect, transmit, or share:

  • Browsing history or page content
  • Personal information of any kind
  • Usage analytics or telemetry

Third-party services

MetVUW Plus operates exclusively on metvuw.co.nz and makes no requests to any third-party servers. It does not load any remote code or resources.

Contact

Questions about this policy can be raised via the extension's source repository.
